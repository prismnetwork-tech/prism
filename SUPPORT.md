# Support

Community support is provided through GitHub Discussions and issues on a
best-effort basis. The open-source project does not provide an uptime,
response-time or compatibility service-level agreement.

Use:

- Discussions for deployment and integration questions.
- Issues for reproducible bugs and scoped feature proposals.
- Private vulnerability reporting for security defects.

Include versions, operating system, sanitized configuration and minimal
reproduction steps. Never include credentials, private keys, wallet secrets,
session tokens or user data.
